Optional filter

  • filter(ctx: SimpleActionContext): boolean
    • Defined in actionMiddlewares/actionTrackingMiddleware.ts:77

    Filter function called before each action starts. If the action is accepted then onStart, onResume, onSuspend and onFinish for that particular action will be called. All actions are accepted by default if no filter function is present.

    Parameters

    • ctx: SimpleActionContext

      Simplified action context.

    Returns boolean

    true to accept the action, false to skip it.
